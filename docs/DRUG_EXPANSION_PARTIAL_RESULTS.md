# Drug Expansion Research — Partial Results (July 3, 2026)

Agent was killed after ~27 min. Four of five sub-agents completed. Cholinergic/NMDA agent did not finish.

## Key finding

**Most approved MS drugs have zero genetic/MR evidence for their specific targets.** Only S1PR1 (fingolimod/siponimod/ozanimod/ponesimod) has any MR support. This means most MS drugs CANNOT be classified by the etiologic-only rule — they lack multi-type etiologic evidence.

This is an honest limitation but also informative: the framework can only classify drugs whose target mechanism has been studied across evidence types.

## MS drugs found (12 total)

All 12 have strong Phase III trial data:
- Anti-CD20: ocrelizumab, ofatumumab, ublituximab (already in catalog)
- S1P modulators: fingolimod, siponimod, ozanimod, ponesimod (S1PR1 has MR support)
- Anti-integrin: natalizumab
- Anti-CD52: alemtuzumab
- Purine analog: cladribine
- Others: teriflunomide, dimethyl fumarate, glatiramer
